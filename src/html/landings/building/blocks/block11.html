«Микрорайон первый»
Разработали максимально простой, яркий и красочный сайт для застройщика спального микрорайона,
где для удобства потенциальных клиентов нами была разработана интерактивная 3D - модель района
с планировками объектов, адаптированный для мобильных устройств интерфейс и внедрена карта строящихся
и сданных объектов в черте города.
